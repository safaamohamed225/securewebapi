# 🔒 Secure API

A simple **secure API** built with .NET Core using **JWT authentication**.

## ✨ Features

- 👤 User registration & login
- 🔑 JWT token authentication
- 🚫 Protected endpoints with role-based access

## 🛠️ Technologies

- ASP.NET Core Web API  
- Entity Framework Core  
- SQL Server  
- JWT (JSON Web Tokens)

## 🚀 Getting Started

1. Clone the repo and open it in Visual Studio or VS Code.  
2. Update the database connection in `appsettings.json`.  
3. Run the API & test endpoints using Postman.
